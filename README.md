# Atividade - Redes Sociais

Este repositório contém uma página web simples em HTML5 estruturada para apresentar um estudo/reflexão sobre o impacto das redes sociais na sociedade moderna, abordando aspectos econômicos, psicológicos e estruturais.

O projeto foi desenvolvido como parte de uma atividade acadêmica para praticar a estruturação de tags HTML essenciais, estilização em linha (*inline* e interna) e criação de tabelas.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação de conteúdo (`<h1>`, `<p>`, listas, `<table>`).
* **CSS3:** Estilização básica para bordas de tabelas (`border-collapse`) e ajuste de tamanhos de fonte (`font-size`).

## 📋 Funcionalidades e Estrutura do Código

A página é dividida de forma linear e limpa nos seguintes blocos:

1.  **Cabeçalho:** Título centralizado informando o tema principal ("As Redes Sociais").
2.  **Introdução:** Breve reflexão sobre o impacto das plataformas na economia e no psicológico das pessoas.
3.  **Lista Ordenada (`<ol>`):** Apresentação das redes sociais mais utilizadas atualmente (Instagram, Facebook, TikTok e Twitter/X).
4.  **Tabela Comparativa (`<table>`):** Exibição de dados organizados contendo o nome da rede, o ano de lançamento e o tipo de conteúdo principal produzido nela.
5.  **Lista Não Ordenada (`<ul>`):** Resumo em tópicos destacando os principais **Impactos Positivos e Negativos** do uso dessas tecnologias.

## 🛠️ Como Executar o Projeto

Para visualizar a página localmente em seu navegador, siga os passos abaixo:

1.  Baixe ou copie o código do arquivo para um documento de texto e salve-o como `index.html`.
2.  Navegue até a pasta onde o arquivo foi salvo.
3.  Dê um duplo clique no arquivo ou clique com o botão direito e selecione **"Abrir com"** -> Escolha o seu navegador de preferência (Google Chrome, Microsoft Edge, Firefox, etc.).
